[[req_nwsviz_items_zarr_endpoint]]
[requirement]
====
[%metadata]
identifier:: /req/nwsviz/items-zarr-endpoint
statement:: The items query SHALL expose a virtual Zarr endpoint for each collection.
part:: Each collection SHALL provide a /collections/{id}/items endpoint.
part:: The items endpoint SHALL include an item with id zarr that acts as a virtual Zarr store.
part:: The zarr item SHALL expose the underlying dataset as a Zarr store accessible via the .zarr/ path convention.
part:: The zarr item SHALL be navigable using standard Zarr clients (e.g. xarray.open_zarr, zarr.open).
part:: The zarr item response SHALL include a link with rel=zarr pointing to the root of the virtual Zarr store.
part:: The virtual Zarr store SHALL reflect the full spatio-temporal extent of the collection.
====
