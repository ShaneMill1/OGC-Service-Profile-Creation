[[req_nwsviz_conformance]]
[requirement]
====
[%metadata]
identifier:: /req/nwsviz/conformance
statement:: The service SHALL declare conformance to OGC API Common, EDR, EDR Part 3, and OGC API Processes.
part:: The /conformance endpoint SHALL include http://www.opengis.net/spec/ogcapi-common-1/1.0/conf/core.
part:: The /conformance endpoint SHALL include http://www.opengis.net/spec/ogcapi-common-1/1.0/conf/json.
part:: The /conformance endpoint SHALL include http://www.opengis.net/spec/ogcapi-common-1/1.0/conf/oas30.
part:: The /conformance endpoint SHALL include http://www.opengis.net/spec/ogcapi-common-2/1.0/conf/collections.
part:: The /conformance endpoint SHALL include http://www.opengis.net/spec/ogcapi-edr-1/1.0/conf/core.
part:: The /conformance endpoint SHALL include http://www.opengis.net/spec/ogcapi-edr-3/1.0/conf/conf-class-core.
part:: The /conformance endpoint SHALL include http://www.opengis.net/spec/ogcapi-processes-1/1.0/conf/core.
====
