[[req_nwsviz_position_coveragejson]]
[requirement]
====
[%metadata]
identifier:: /req/nwsviz/position-coveragejson
statement:: The position query SHALL return CoverageJSON.
part:: Each collection SHALL provide a /collections/{id}/position endpoint.
part:: The position endpoint SHALL accept a coords parameter containing a WKT POINT geometry.
part:: The position endpoint SHALL return a valid CoverageJSON Coverage object as its default response.
part:: The response Content-Type SHALL be application/prs.coverage+json.
====
